# PetsLeaderboard
This app features the SwiftUI Layout protocol and Grid View. The app
uses a leaderboard system to demonstrate the aforesaid features.
## Featuring
- Layout Protocol
- Grid View
- LayoutValueKey Type
- SwiftData
